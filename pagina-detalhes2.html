Sensor de Temperatura, Pressão atmosférica e Aceleração

Descrição e Utilidade

Este projeto utiliza a placa Arduino MKR WiFi 1010 equipada com o módulo IoT Carrier. Ele permite a leitura de temperatura, pressão atmosférica e aceleração do ambiente. É ideal para monitoramento climático, prototipagem de IoT e experimentos educacionais.

"O projeto foi criado para facilitar a compreensão de sensores integrados ao IoT Carrier, além de permitir visualização imediata dos dados em um display onboard e em uma futura integração online."

Leitura de Sensores

Temperatura

Mostra a temperatura atual do ambiente em Celsius.

Pressão Atmosférica

Permite monitorar a pressão atmosférica em hPa, útil para previsões meteorológicas simples.

Aceleração

Fornece valores em tempo real, permitindo o monitoramento.

Gráfico

Exibe os dados dos sensores em tempo real, permitindo uma análise visual das informações coletadas.
